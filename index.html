HTML CSS

HTML

< h1 - h6>< /h1 - h6> Создает заголовок h1 самый большой h6 самый маленький

< p >< /p > Создает параграф

< br > line breaker все что после этого тега будет перенесено на новую строку

< hr > horizontal rule создает горизонтальную линию

< !-- smth -- > создает комментарий

< em >< em > делает текст между этих тегов курсивным (вместо em - emphasis можно использовать i - italianic)

< strong >< strong > делает текст жирным

< mark >< mark > делает текст как будто выделенным хайлайтером (цвет можно изменить с помощью css)

< li >< li > создает список вещей li - list item

< ul >< ul > создает неупорядочный список ul - unordered list

< ol >< ol > создает упорядочный список ol - ordered list

< img src="" alt="" > создает фото в src пишем его url или любой путь к нему в alt информацию которая отобразится если фото не прогрузится

table, thead, tbody , tfoot, tr - table row , th - table headings and td - table data; Создают таблицу

< a href="#">< /a> Создает ссылку в href вставляем ссылку на что то, так же можно написать #+id другого элемента и тогда будет internal link

< abbr title="Расшифровка аббревиатуры"> Аббревиатура< /abbr> > Создает аббревиатуру и подчеркивает ее точками снизу в тайтл пишем расшифровку аббревиатуры

< button type="button" name="button">< /button> Создает кнопку

< span >< /span > Обычно используется для выделения части из полноценого тега

< div >< /div > объединяет элементы в блок

< input >input очень обширная тема в HTML лучше посмотрите здесь W3school

CSS

< link rel="stylesheet" href="/css/master.css" > (точно не знаю как она работает я использкю link только что бы сделать external css)

color:rgb(0,0,0,) or #fff or "red", "blue" - изменяет цвет текста

background-color тоже как и color но изменяет задний фон

border создает границу вокруг элемента лучше всего писать border:5px solid red (размер стиль цвет) об этом оч много и лучше чекнуть в инете

heigth and width определяют длину и ширину обьекта

Box model у каждого обьекта есть своя определненная box model а то есть это место которое занимает обьект на вашем вебсайте

если вы нажмете в браузере просмотреть код и наведетесь на какой то элемент вам покажется box model вообще она состоит из состовляющего padding border margin outline

padding это пространство между границой и состовляющем вашего обьекта увеличение padding увеличивает box model и сам элемент

margin пространство после границы изменяет box model но не изменяет размер самого элемента

outline это такая же граница как бордер но после margin; она имеет такие же свойства

нажмите ПКМ - просмотреть код и посмотрите на box model

font-family;font-style;font-size нужны для изменения текста

list-style-type: circle;square;upper-roman;lower-alpha изменяют стиль ul и ol

display:inline,block,none влияют на box model что бы узнать все изменения лучше посмотреть здесь
